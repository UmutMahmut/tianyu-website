{% if is_zh %}项目概述{% else %}Project Overview{% endif %}

{% if is_zh %} 天语计划（Tianyu Project, JUST-P）是一个面向时间域天文学和系外行星搜索的大视场巡天项目，采样频率可配置覆盖 0.3 秒至 1 周。 该计划名称来源于李政道先生提出的“以天之语，解物之道”之哲学理念，旨在通过高时间分辨率的长时间基线观测， 发现类太阳系的长周期冷气体巨行星，并开展覆盖多时标的光变与瞬变天体研究。 {% else %} The Tianyu Project (JUST-P) is a wide-field photometric survey for time-domain astronomy and exoplanet detection, with configurable cadence from 0.3 s to 1 week. Inspired by Tsung-Dao Lee’s philosophy “From the Language of Heaven to the Rationale of Matter,” it aims to detect long-period, cold gas giants in solar-like systems, and to study variability and transients across a broad range of timescales. {% endif %}

{% if is_zh %}主要特色{% else %}Key Features{% endif %}

    {% if is_zh %}
  • 望远镜系统：1 米口径广角望远镜，约 10 平方度视场
  • 采样频率：0.3 秒–1 周（可配置），支持长时间基线监测
  • 多目标覆盖：目标天体数量约 1 亿，覆盖 TESS 连续观测区与 Kepler/K2 区域
  • 多时标科学：从系外行星到变星与瞬变源，覆盖多时标光变研究
    • {% else %}
  • Telescope system: 1-m wide-field telescope, ~10 deg² FoV
  • Cadence: 0.3 s–1 week (configurable), enabling long-term monitoring
  • Massive target coverage: ~100M targets in the TESS CVZ and Kepler/K2 regions
  • Broad science: From exoplanets to variability and transients across timescales
    • {% endif %}

{% if is_zh %}科学目标{% else %}Science Objectives{% endif %}

    {% if is_zh %}
  • 发现类太阳系中冷巨行星的凌日信号
  • 研究超新星、快蓝光瞬变（FBOT）、潮汐破坏事件等瞬变源
  • 监测近地天体（NEO）、彗星与 TNO 掩星等小天体活动
  • 寻找伽马暴（GRB）与快速射电暴（FRB）的光学对应体
    • {% else %}
  • Detect transits of long-period cold giants in solar analogs
  • Study transients like supernovae, FBOTs, and TDEs
  • Track small bodies such as NEOs, comets, and occultations
  • Search for optical counterparts of GRBs and FRBs
    • {% endif %}
